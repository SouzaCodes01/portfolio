# 🎨 Portfolio — SouzaCodes01

Portfolio pessoal moderno construído com **React**, **TypeScript**, **TailwindCSS** e **Framer Motion**.

## ✨ Features

- 🎭 **Animações suaves** com Framer Motion
- 🌙 **Dark/Light mode** com transição animada
- 📱 **100% Responsivo** — Mobile-first design
- ⚡ **Blazing fast** — Vite + Code splitting
- ♿ **Acessível** — Semântica HTML e ARIA labels
- 🔍 **SEO otimizado** — Meta tags e Open Graph
- 📧 **Formulário de contato** funcional com EmailJS
- 🎵 **Seção Vibe Coding** — Showcase da metodologia

## 🖼️ Seções

| Seção | Descrição |
|-------|-----------|
| **Hero** | Apresentação com animação de typing |
| **Sobre** | Bio, skills e experiência |
| **Tech Stack** | Grid interativo de tecnologias |
| **Projetos** | Cards com filtro por categoria |
| **Vibe Coding** | Minha abordagem com AI-assisted dev |
| **Contato** | Formulário + links sociais |

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/SouzaCodes01/portfolio.git
cd portfolio

# Instale
npm install

# Rode
npm run dev

# Build
npm run build
```

## 🏗️ Estrutura

```
src/
├── assets/           # Imagens e ícones
├── components/
│   ├── layout/       # Navbar, Footer
│   ├── sections/     # Hero, About, Projects, Contact
│   ├── ui/           # Button, Card, Badge, Modal
│   └── animations/   # Wrappers de animação
├── data/             # Conteúdo estático (projetos, skills)
├── hooks/            # Custom hooks
├── styles/           # Global styles e tema
├── types/            # TypeScript types
├── utils/            # Helpers
├── App.tsx
└── main.tsx
```
